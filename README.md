1. Слияние списков (ListMerger)

Что делаем: объединяем списки L1 и L2 в один.

Как реализовано:

return L1.Concat(L2).ToList();

Используется Concat для последовательного добавления элементов второго списка к первому.


2. Подсчёт элементов с одинаковыми соседями (ListNeighbors)

Что делаем: ищем элементы, у которых левый и правый сосед равен самому элементу.

Как реализовано:

for (int i = 1; i < L.Count - 1; i++)
    if (L[i - 1] == L[i] && L[i] == L[i + 1])
        count++;


Цикл начинается с 1 и заканчивается на Count - 1, чтобы исключить крайние элементы без двух соседей.

4. Анализ текста и уникальные согласные (CharFinder)

Что делаем: выводим согласные буквы русского алфавита, встречающиеся ровно в одном слове файла.

Почему: задание на работу с текстом, фильтрацию и подсчёт уникальных элементов.

Как реализовано:

Считываем текст из файла с кодировкой Windows-1251.

Разбиваем на слова по пробелам и знакам препинания.

Для каждого слова выбираем уникальные согласные.

Объединяем все согласные из слов, группируем по букве, выбираем те, что встречаются ровно в одном слове.

Сортируем и выводим результат.

var uniqueConsonants =
    words.Select(word =>
            word.ToLower()
                .Where(c => RussianConsonants.Contains(c))
                .Distinct()
    )
    .SelectMany(consonants => consonants)
    .GroupBy(c => c)
    .Where(g => g.Count() == 1)
    .Select(g => g.Key)
    .OrderBy(c => c)
    .ToList();


Интерфейс: выводит количество слов и список согласных, встречающихся ровно в одном слове.

Примеры работы программы

Слияние списков:

Результат слияния: 1 3 5 2 4 6


Элементы с одинаковыми соседями:

Список: 1 2 2 2 3 4 4 4 5

Количество элементов с равными соседями: 2


Уникальные согласные в тексте:
